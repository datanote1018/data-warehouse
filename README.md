# Data Warehouse Knowledge Base

An open-source knowledge base for data warehouse engineers — covering SQL practice, naming conventions, development standards, and more.

**Live Site:** [https://datanote1018.github.io/data-warehouse/](https://datanote1018.github.io/data-warehouse/)

## Contents

| Module | Description | Status |
|--------|-------------|--------|
| [SQL Tutorial](sql-tutorial/) | Interactive HiveSQL tutorial with exercises and built-in query editor | ✅ Ready |
| [Standards](standards/) | Data warehouse layer design, development guidelines | 🚧 In Progress |
| [Glossary](glossary/) | Common word roots and naming patterns for DW table/field design | 🚧 In Progress |

## Quick Start

No build tools needed. Just open `index.html` in a browser, or visit the live site above.

```bash
git clone https://github.com/datanote1018/data-warehouse.git
cd data-warehouse
open index.html
```

## Contributing

We welcome contributions! Please read [CONTRIBUTING.md](CONTRIBUTING.md) before submitting a PR.

**Quick summary:**
1. Fork the repo and create a branch from `master`
2. Make your changes following the [Style Guide](docs/style-guide.md)
3. Submit a PR — all changes require at least one review before merging
